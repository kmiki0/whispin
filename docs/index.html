特徴

なぜ Whispin か。

文脈で補正

画面の固有名詞・専門用語を手がかりに誤変換を修正。

押して話すだけ

長押し→離すだけ。元アプリへ自動で貼り付け。

コストはゼロ

Groq 無料枠で完結。クレカ不要のキーだけ・¥0。

軽量・常駐

Tauri 2 製。常駐し必要時だけマイクを起動。

プライバシー配慮

キーは暗号化保存。画面文脈の送信はオフ可。

邪魔しないUI

画面上端の波形ノッチで状態がひと目で分かる。

FAQ

よくある質問。

対応OSは？
Windows 11。常駐するネイティブアプリ（MSIインストーラーで配布予定）です。
料金は？
¥0。Groq 無料枠で動作。クレカ不要のキーを貼るだけ。
プライバシーは？
キーは暗号化保存。画面文脈の送信はオフ可。音声・画面はため込みません。
ホットキーは変えられる？
設定でキー／マウスボタンを自由に変更可。既定は右クリック長押し。
どのアプリに入力できる？
直前のウィンドウへ自動貼り付け。入力欄があればどこでも。
