Whispin は、いま見ている画面の会話文脈を手がかりに、固有名詞や同音異義語の誤認識を補正する日本語音声入力ツール。 Tauri 製で軽く、Groq の無料枠で動くのでコストはゼロ。

なぜ Whispin?

ブラウザ標準の音声入力では拾いきれない、日本語ならではの誤認識を解決します。

🎯

文脈で補正

画面に映る固有名詞や専門用語を手がかりに、同音異義語・誤変換を文脈で正します。Claude Code / claude.ai のようなワークフローに最適。

⚡

押して話すだけ

ホットキーを長押しして話し、離すと文字起こし。クリップボード経由で元のウィンドウへ自動で貼り付け。マウス操作は不要。

💰

コストはゼロ

Groq の無料枠で動作。ASR は Whisper Large v3 Turbo。クレカ不要の API キーを発行するだけで使い始められます。

🪶

軽量・常駐

Tauri 2 製のネイティブアプリ。バックグラウンドに常駐し、必要なときだけマイクを起動。リソースを食いません。

🔒

プライバシー配慮

API キーは Windows DPAPI で暗号化して保存。AI校正に使う「画面文脈」の送信はワンクリックでオフにでき、音声・画面はローカルにため込みません。

🌊

邪魔しないUI

画面上端にスッと現れる波形オーバーレイで、録音状態がひと目で分かる。作業の邪魔をしません。

使い方は 3 ステップ

セットアップが終われば、あとは押して話すだけ。

1

長押しで録音

トリガー（既定は右クリック長押し、設定で変更可）を押している間だけ録音。push-to-talk 方式なので誤起動しません。

2

離すと文字起こし

キーを離すと録音停止。Whisper で文字起こしし、画面の文脈をもとに誤認識を補正します。

3

自動で貼り付け

結果をクリップボードに書き込み、元のフォーカスウィンドウへ戻って自動入力。すぐ続きが書けます。
