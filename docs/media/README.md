# ランディングページ用メディア素材

`docs/index.html`（ターミナル案）で使う画像・動画の置き場所です。下のファイル名で
このフォルダに置くと、ページのプレースホルダと差し替えます。

| ファイル | 用途 / 配置 | 内容 | 推奨仕様 |
|---|---|---|---|
| `demo.mp4` | ヒーローのメイン | ホットキー長押し → 発話 → 元アプリへ自動貼り付け（補正込み）の一連 | 16:9・1280×720 以上・H.264 mp4・15〜30秒・音声不要（自動ループ/ミュート再生）・できれば 5MB 前後 |
| `demo-poster.png` | `demo.mp4` の静止画 | 動画読み込み前に表示するサムネ（最初のフレームでOK） | 16:9・1280×720 |
| `notch.png` | 実際の画面 ① | 録音中の波形オーバーレイ（画面上端のノッチ）。上端を中心にトリミング | 16:9・横1200px 以上・PNG |
| `correction.png` | 実際の画面 ② | 補正の before/after 実例（Claude Code 等に貼り付いた結果） | 16:9・横1200px 以上・PNG |
| `settings.png` | 実際の画面 ③ | 設定ウィンドウ（トリガー / AI校正 / 辞書 タブのいずれか代表1枚） | 16:9・横1200px 以上・PNG |

## 撮り方メモ
- **スクショ**: Windows は `Win + Shift + S`（範囲指定）。設定ウィンドウはトレイの Whispin →「設定」から。
- **ノッチ**: 録音中（トリガー長押し中）に表示されます。または起動直後の「Whispin Ready」表示中でも可。
- **デモ動画**: Xbox Game Bar（`Win + G`）や OBS で画面録画 → mp4 書き出し。GIF でも可（`demo.gif`）。
- 余計な個人情報が映らないようご注意ください。

素材が揃ったら、`<img>` / `<video>` をページに組み込み、`docs/index.html` を確定します。
